# Treeco - Python Interface

[![Python 3.8+](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)

The Python interface to `treeco` can be installed using [pip](https://pypi.org/project/pip/) as follows:
```bash
pip install git+https://github.com/TheoGuyard/treeco.git#subdirectory=python
```
Check out the [python/examples](python/examples) folder for a quick start guide.
